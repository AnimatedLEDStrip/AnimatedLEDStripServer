addAnimation

fun addAnimation(params: AnimationData): Unit

Adds a new animation.

If animation.continuous is false: Runs the animation in a new thread once.

If animation.continuous is true: Creates a ContinuousRunAnimation instance in a new thread, Adds pair with the animation ID and ContinuousRunAnimation instance to continuousAnimations.

Parameters

params - An AnimationData instance containing data about the animation to be run
